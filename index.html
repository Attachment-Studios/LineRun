LineRun is a game engine which is not that complex how one would imagine... Actually this engine was made for no purpose, but with time the devlopers still don't know why or why not to use this engine or even make it. Seriously talking, it is not that bad, but as it was made by the developer at the age of 16 or 17, so it might get complex. Still not bad though. There is also a wiki for the engine describing it's various commands.

Download

Examples

Simple "Hello World"

log.text(Hello World)

Commenting

-->$ Comment

Making Variables

var.num(myNumericVariable) -->$ makes numeric variable
var.lgc(myLogicVariable) -->$ makes logic variable
var.str(myStringVariable) -->$ makes string variable

Print Variable Values

var.getVal(var) -->$ prints the value of variable var
